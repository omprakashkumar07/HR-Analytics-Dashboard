# HR Analytics Dashboard

This project focuses on analyzing employee attrition using Excel and Power BI.

## Objective
To identify key factors affecting employee attrition and generate insights using data visualization.

## Tools Used
- Microsoft Excel
- Power BI

## Key Insights
- Higher attrition observed in lower salary groups
- Maximum attrition in early career employees (0–3 years)
- Job role significantly impacts employee retention
